Node.js (Express) + React (Vite) + MongoDB (Cloud Atlas) setup on a virtual server.

  • An active AWS Account.
  • Git Bash (for Windows) or Terminal (macOS/Linux).

🚀 Phase 1: Setting up the Virtual Machine

Step 1: Launch the EC2 Instance

Why we do this:

We rent a remote virtual computer in the cloud running 24/7. Choosing Ubuntu Server provides a lightweight, standard Linux OS to run Node and host React.

  • Open the AWS Console, search for EC2, and click Launch Instance.
  • Name: balor-salon-server
  • OS Image: Ubuntu (Ubuntu Server 24.04 LTS, Free tier eligible).
  • Instance Type: t3.micro or t2.micro.
  • Key Pair: Select ujjwal_domain (or create a new one).
  • Firewall Settings: Allow **SSH**, **HTTP**, and **HTTPS** traffic from anywhere.
  • Storage: Keep default 8 GiB.
  • Click Launch and copy the Public IPv4 IP.

Step 2: Connect to the Server via SSH

Why we do this:

Connects your local machine's terminal securely to the cloud server's terminal.

Open Git Bash in your project directory and run:

ssh -i ./ujjwal_domain.pem ubuntu@35.154.20.254

Step 3: Prepare the Server Environment

Why we do this:

Updates packages and installs Node.js runtime and PM2 process manager so your backend runs 24/7.

# Update packages
sudo apt update

# Fetch Node v20 repository
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -

# Install Node.js
sudo apt install -y nodejs

# Install PM2 Process Manager
sudo npm install -g pm2

🌐 Phase 3: Routing Traffic with Nginx

Step 7: Install and Configure Nginx

Why we do this:

Configures a web server to listen for requests to balor.ujjwalchoubey.me, serve frontend HTML/CSS files directly, and proxy all /api calls to Node.js on port 5000.

# Install Nginx
sudo apt install -y nginx

# Edit Configuration
sudo tee /etc/nginx/sites-available/default << 'EOF'
server {
    listen 80;
    server_name balor.ujjwalchoubey.me www.balor.ujjwalchoubey.me;

    root /home/ubuntu/Balor-Smart_Grooming_and_Salon_Platform/frontend/dist;
    index index.html;

    location / {
        try_files $uri $uri/ /index.html;
    }

    location /api {
        proxy_pass http://localhost:5000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
EOF

# Give access and restart Nginx
chmod 755 /home/ubuntu
sudo nginx -t
sudo systemctl restart nginx

🛡️ Phase 4: Persistence & Updating the App

Step 8: Persistent Process Setup

pm2 startup
# (Run the command generated in the terminal, then save)
pm2 save

Step 9: Pushing Code Updates

To deploy changes in one command, create a deploy.sh script inside /home/ubuntu/Balor-Smart_Grooming_and_Salon_Platform:

#!/bin/bash
git pull
cd backend && npm install && pm2 restart balor-backend
cd ../frontend && npm install && npm run build
echo "Deploy Complete!"

Run using ./deploy.sh.
